# Salah Buddy - Ramadan Platform for Kids

## Project Overview

Salah Buddy is a free Ramadan web platform for kids featuring:
- User authentication (kids/family accounts)
- Salah (prayer) tracking
- Fasting tracking
- Personal dashboard
- Competition/leaderboard dashboard

## Target

- Quick to build: 2-3 weeks to MVP
- Free or very low cost during Ramadan (30 days)
- Scalable for Ramadan traffic
- Kid-friendly UI/UX
