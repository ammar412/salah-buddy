Paula Scher's Typography Principles
Applied to Salah Buddy
01
Big & Bold Wins
Don't be timid with type. Kids respond to SCALE. Prayer names at 32px, stats at 72px, celebrations at 96px. Let typography create the excitement.
02
Weight Expresses Emotion
Light (300) whispers. Regular (400) speaks. Bold (700) shouts. Black (900) celebrates. Use weight to create vocal variety in your visual voice.
03
Hierarchy = Clarity
Kids shouldn't have to search for what matters. Prayer names? Bold at 32px. Time? Medium at 18px. Stats? Black at 72px. Clear hierarchy means instant comprehension.
04
Space Amplifies
Generous spacing around big type makes it BIGGER. 48px padding around a 72px number creates breathing room that amplifies impact.
05
Contrast Creates Energy
Gold on cream. Dark text on white. 900-weight next to 400-weight. Strong contrast = visual energy that matches kids' actual energy.
06
Numbers Are Heroes
Stats are achievements. Render them at display sizes (72-96px) with Black weight (900). Make every "5/5 prayers" feel like winning a trophy.

Implementation Checklist

Typography Setup

1. Font Loading 

<link href="https://fonts.googleapis.com/css2?family=Lexend:wght@300;400;500;600;700;800;900&display=swap" rel="stylesheet">

2. CSS Variables (Tailwind Config) 

:root {
  /* Sizes */
  --type-hero: 96px;
  --type-display: 72px;
  --type-headline: 48px;
  --type-title: 32px;
  --type-body-lg: 24px;
  --type-body: 18px;
  --type-caption: 14px;

  /* Weights */
  --weight-light: 300;
  --weight-regular: 400;
  --weight-medium: 500;
  --weight-semibold: 600;
  --weight-bold: 700;
  --weight-extrabold: 800;
  --weight-black: 900;
}

3. Component Classes 

.greeting-name {
  font-size: var(--type-display);
  font-weight: var(--weight-black);
  color: var(--gold);
  line-height: 1;
}

.prayer-name {
  font-size: var(--type-title);
  font-weight: var(--weight-bold);
  line-height: 1.2;
}

.stat-number {
  font-size: var(--type-display);
  font-weight: var(--weight-black);
  color: var(--gold);
}
Final Recommendation

Typography isn't just about readability—it's about creating an emotional experience. For Salah Buddy, Lexend at scale creates warmth, clarity, and celebration that matches Luqmay's educational mission.

Every prayer logged should feel like a victory. Every stat should feel like a trophy. Big, bold, expressive typography makes kids WANT to engage—because the interface itself is celebrating with them.
